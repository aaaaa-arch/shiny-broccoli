# XY脚本中心

> 一个面向 Roblox 的 Lua 脚本集合，专注游戏自动化与功能增强

## 📦 脚本列表

| 脚本 | 适用游戏 | 功能简介 | 版本 |
|------|---------|---------|------|
| 钓鱼模拟器 | 钓鱼模拟器 | 自动抓捕、自动售卖、每日宝箱、升级、重生、买蛋 | v2.0 |
| 犯罪 | 犯罪模拟器 | 夜视、自动开门、拾取、穿墙、近战光环、ESP | v1.03 |
| 力量传奇 | 力量传奇 | 自动锻炼、跑步机、传送、修改属性、自动宝箱 | v2.0 |
| 99夜 | 森林中的99夜 | 无限血、杀戮光环、自动砍树、种植、捕兽夹、图案玩法 | v2.0 |
| 请捐赠 | 通用 | 摊位管理、自动感谢、防AFK、自动说话 | v1.0 |

## 🚀 使用方法

1. 复制对应脚本的 `.lua` 文件内容
2. 打开 Roblox，注入你的执行器
3. 粘贴并执行
4. 等待 UI 加载完成即可使用

## ⚙️ 通用依赖

部分脚本依赖以下 UI 库（已内置自动加载）：

- **WindUI** - 现代化 UI 框架
- **Zephyr UI** - 钓鱼模拟器专用界面

## ⚠️ 注意事项

- 脚本仅供学习交流，请勿用于破坏游戏平衡
- 部分功能可能因游戏更新失效，欢迎反馈
- 使用前请确保执行器支持 `loadstring` 和 `HttpGet`
